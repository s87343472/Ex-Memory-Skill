A Claude skill for processing memories and emotions around an ex.

## Two modes

### Simulate

Upload chat logs, screenshots, or just describe your ex — Claude steps into their voice and speaks the way they did. Say what you never got to say, or just feel that presence one more time.

### Reflect

Not ready to simulate? Just want to talk? Claude will sit with you and help you make sense of the relationship — the patterns, the feelings, what you're carrying. No judgment, no "just move on", no tidy conclusions.

## Installation

1. Download `前任.skill.md` from this repo
2. Go to [Claude.ai](https://claude.ai) → Settings → Skills → Import Skill
3. Upload the file, done

## How to trigger

Just tell Claude what's going on — the skill activates automatically. No special command needed.

**Enter [Simulate](#simulate):**

```
I want you to be my ex. Their name was Jay, quiet, always replied with one word…
Can you be them? I'll send you some of our old chats.
```

**Enter [Reflect](#reflect):**

```
I don't want to simulate anything, I just want to talk through it
Help me figure out what I actually learned from this relationship
```

**Not sure yet:**

```
I just broke up with someone
I keep thinking about my ex
There's this person I used to date…
```

Just start talking — Claude will be with you first, then ask how you'd like to continue.

## Supported inputs

The more you share before Simulate mode, the more accurate the result. Any combination works:

| Input | Description |
|---|---|
| Chat screenshots | Upload images directly — Claude reads speech patterns, tone, habits |
| Text description | Their personality, catchphrases, how they spoke to you, how they fought |
| Conversation excerpts | Paste memorable or representative exchanges |
| Relationship context | How you met, how long, how it ended |

Even a sentence or two is enough to start — Claude fills in the gaps as you talk.

---

## Design philosophy

**Authenticity over comfort** — In Simulate mode, Claude stays true to who they were, even when that's uncomfortable. No invented warmth.

**You set the pace** — No one's telling you to move on. Take as long as you need — or don't.

**Crisis care** — If Claude senses you're in real distress, it will gently step out of character to check on you.

---

## Notes

This skill is not a substitute for professional mental health support. If you're in serious distress, please reach out to a professional.
