## 两种模式

### 模拟模式

把聊天记录、截图、或者你对 TA 的描述发给 Claude，Claude 会扮演你的前任，用 TA 的说话方式和你对话。说那些当时没说出口的话，或者只是想再感受一下那个人的语气。

### 反思模式

不想模拟，只想聊聊这段感情——Claude 会陪你慢慢梳理你们的相处模式、你在这段关系里的感受，以及你可以带走的东西。不评判，不催你走出来，不给标准答案。

## 安装方法

1. 下载本 repo 里的 `前任.skill.md` 文件
2. 打开 [Claude.ai](https://claude.ai) → Settings → Skills → Import Skill
3. 上传文件，完成

## 怎么触发

直接和 Claude 说你的情况就好，skill 会自动识别并激活，不需要特别的指令。

**进入[模拟模式](#模拟模式)：**

```
我想让你扮演我前任，TA 叫小宇，话不多，喜欢用"嗯"回我…
你可以扮演 TA 吗？我来给你发一些我们的聊天记录截图
```

**进入[反思模式](#反思模式)：**

```
我不想模拟，就想聊聊这段关系
帮我复盘一下，我在这段感情里学到了什么
```

**不确定想要什么：**

```
我刚分手
我最近总是想起一个人
我有个前任…
```

直接说就好，Claude 会先陪你聊，再问你想怎么继续。

## 可以提供哪些材料

进入模拟模式之前，材料越多，还原越准。支持任意组合：

| 材料 | 说明 |
|---|---|
| 聊天记录截图 | 直接上传图片，Claude 分析说话风格和用词习惯 |
| 文字描述 | TA 的性格、口头禅、对你的称呼、争吵时的样子 |
| 对话片段 | 粘贴几段有代表性的聊天内容 |
| 关系背景 | 你们怎么认识、在一起多久、怎么分开的 |

只有一两句描述也可以开始，Claude 会在对话中慢慢补全。

## 设计理念

**真实 > 治愈** — 模拟模式里宁可有一点难受，也不造假的温柔。TA 不是完美的，还原出来的也不该是。

**你决定节奏** — 没有人催你走出来，没有人告诉你应该放下了。你准备好了再说，或者永远不说，都可以。

**危机关怀** — 如果 Claude 感知到明显的情绪危机信号，会温柔地退出扮演角色，先关心你这个人。

这个 skill 不是心理咨询或治疗的替代品。如果你正在经历严重的情绪困扰，请寻求专业帮助。
